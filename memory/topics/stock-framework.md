## 一、七维结构（新增第7维：情感雷达）

### 1. 宏观周期（Macro Cycle）
- 利率/流动性方向
- 中美政策周期位置
- 汇率与跨境资本流动

### 2. 行业景气（Industry Prosperity）
- 产能周期（过剩/出清/紧缺）
- 价格走势与库存水平
- 技术迭代与替代威胁

### 3. 公司质地（Company Quality）
- ROE/现金流/负债结构
- 管理层诚信与战略执行力
- 护城河宽度（成本/技术/品牌/牌照）

### 4. 业绩动能（Earnings Momentum）
- 营收/利润增速趋势
- 毛利率变化与费用控制
- 订单/产能释放能见度

### 5. 估值水位（Valuation）
- PE/PB/PS 历史分位
- DCF 隐含回报 vs 无风险利率
- 同业比较与海外对标

### 6. 技术结构（Technical Structure）
- 多周期均线排列（日/周/月）
- 成交量与筹码分布
- 关键支撑/压力位与形态

### 7. 情感雷达（Sentiment Radar）⭐ 新增
> 基于 LLM+新闻情感的大宗商品/股票预测研究（浙大&比勒费尔德，2026-04）：
> 高波动市场下，融合新闻情感的策略夏普比从 0.23 → 1.04（+359%）。

#### 7.1 波动率分层权重
| 市场环境 | 情感雷达权重 | 其他六维权重 | 核心逻辑 |
|---------|-------------|-------------|---------|
| **高波动**（地缘冲突、政策急转、黑天鹅） | **40-50%** | 50-60% | 传统模型失效，情绪决定短期方向 |
| **中等波动**（趋势行情、结构性牛市） | **30-40%** | 60-70% | 情感信号独立性强，数据噪音需过滤 |
| **低波动**（震荡磨底、成交低迷） | **10-20%** | 80-90% | 情感信号趋同，回归基本面定价 |

#### 7.2 情感信号三要素
1. **来源质量**（信噪比决定预测力）
   - ✅ 高权重：路透社、France24、彭博、公司官方披露
   - ⚠️ 中权重：券商研报、行业协会数据
   - ❌ 低权重/过滤：中文自媒体、股评节目、论坛情绪

2. **话题筛选**（8个有效话题 vs 4个噪音话题）
   - ✅ **有效**：公司新闻、供应中断、库存、价格走势、需求展望、地缘政治、宏观经济、其他
   - ❌ **噪音**：产量（太快被消化）、市场分析（事后诸葛亮）、贸易政策（预期过长）、环境（短期无关）

3. **事件类型**（事实 > 前瞻）
   - ✅ **事实报道**：产量数据发布、库存报告、合同签订、政策落地 → 夏普比 0.62+
   - ❌ **前瞻预测**：分析师目标价、公司指引、媒体渲染 → 夏普比 ≈ 0（已price in）

#### 7.3 情感打分规则（月度/周度跟踪）
- **+2 强烈正面**：多个高信源同时确认需求爆发/政策利好/库存紧张
- **+1 正面**：主流信源偏向乐观，且有事实支撑
- **0 中性**：信息混杂，或处于数据真空期
- **-1 负面**：主流信源偏向悲观，供应过剩/需求走弱信号出现
- **-2 强烈负面**：黑天鹅确认，情绪极度恐慌（往往是短期逆向买点）

## 二、共振评分卡

每维满分 100，按市场环境动态加权：

```
总分 = Σ(维度得分 × 维度权重)

默认权重（低波动期）：
宏观15% + 行业15% + 公司15% + 业绩15% + 估值15% + 技术15% + 情感10%

高波动期调整：
宏观15% + 行业10% + 公司10% + 业绩10% + 估值10% + 技术10% + 情感35%
```

## 四、操作纪律

1. **90+分**：重仓，设好止损
2. **70-80分**：轻仓试探，等业绩/情绪催化
3. **50-60分**：观望，不做左侧
4. **<50分**：回避，宁可错过
5. **止损是生命线**：单票亏损-8% 无条件减仓，-15% 强制止损
